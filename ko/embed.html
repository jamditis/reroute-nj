웹사이트에 Reroute NJ를 넣으세요

Reroute NJ는 무료 오픈소스 커뮤니티 도구입니다. 뉴스룸, 출판사, 커뮤니티 단체 등 누구나 도구를 임베드하거나 링크하거나 재게시할 수 있습니다.

임베드 코드 생성기

iframe 임베드 코드를 복사하여 웹사이트에 Reroute NJ 도구를 삽입하세요. 도구는 iframe 내에서 완전히 인터랙티브하고 반응형으로 작동합니다.

복사됨!

뉴스룸 및 출판사를 위한 안내

Reroute NJ는 기자, 뉴스룸, 커뮤니티 출판사가 활용할 수 있도록 설계되었습니다. 사용 방법은 다음과 같습니다.

기사에 임베드하기

  • 위의 임베드 코드 생성기에서 iframe 코드를 복사하세요.
  • CMS(WordPress, Ghost, Squarespace 등)에 붙여넣으세요 — 대부분 원시 HTML 블록을 지원합니다.
  • 임베드는 반응형이며 모바일에서도 작동합니다.
  • API 키나 계정이 필요 없습니다. 바로 사용할 수 있습니다.

기사에서 링크하기

  • Portal Bridge 기사에 "통근 계획하기" 링크 박스를 추가하세요.
  • 기사 내용에 맞는 특정 도구에 링크하세요.
  • 소셜 미디어 미리보기를 위한 OG 이미지와 소셜 카드가 이미 설정되어 있습니다.
  • 추천 앵커 텍스트: "무료 Reroute NJ 도구로 통근을 계획하세요."

기사 제보

  • 뉴스 페이지에서 Portal Bridge 관련 보도를 모아두고 있습니다.
  • 기사를 제안하려면 URL과 세부 정보를 포함하여 GitHub 이슈를 등록하세요.
  • 또는 amditisj@montclair.edu로 제목에 "Reroute NJ coverage"를 넣어 이메일을 보내세요.
  • 오피니언, 분석, 커뮤니티 보도 등 관련 기사를 모두 포함합니다.

공동 브랜딩 및 맞춤 설정

자체 브랜딩으로 Reroute NJ를 운영하고 싶으신가요? 오픈소스이므로 가능합니다.

옵션 1: 프로젝트 포크

1

GitHub 저장소를 조직 계정으로 포크하세요.

2

css/styles.css를 편집하여 색상, 폰트, 브랜딩을 변경하세요. 모든 테마 값은 :root 블록의 CSS 커스텀 속성입니다.

3

브랜드 이름, 로고, 하단 크레딧을 교체하세요. 원하면 뉴스룸 헤더/내비게이션을 추가할 수 있습니다.

4

GitHub Pages, Netlify, Vercel 또는 자체 서버에 배포하세요. 빌드 과정이 필요 없습니다 — 정적 HTML입니다.

옵션 2: 자체 소개와 함께 임베드

1

위의 iframe 코드로 도구를 임베드하세요.

2

iframe 주변에 자체 헤더, 소개 텍스트, 브랜딩을 추가하세요.

3

주변 페이지를 출판물 디자인에 맞게 스타일링하세요.

기여 방법

Reroute NJ는 커뮤니티 프로젝트입니다. 기술 수준에 관계없이 도움을 줄 수 있습니다.

비기술적 기여

  • 기사 제안: 빠뜨린 Portal Bridge 기사를 알고 계신가요? 이슈를 등록하거나 joe@amditis.com으로 이메일을 보내세요.
  • 오류 신고: 시간표 데이터나 환승 소요 시간이 잘못된 경우 알려주세요.
  • 도구 공유: 통근하는 지인에게 알려주고, 소셜 미디어에 공유하고, 동네 커뮤니티에 올려주세요.
  • 번역: NJ Transit 이용자가 사용하는 스페인어 및 기타 언어로 번역을 도와주세요.

기술적 기여

  • 버그 신고: 버그를 발견하셨나요? GitHub에서 이슈를 등록하세요.
  • 풀 리퀘스트 제출: 코드베이스는 빌드 과정 없는 순수 HTML/CSS/JS입니다. 포크하고 수정하고 PR을 보내세요.
  • 데이터 추가: data/coverage.json에 새 기사를 업데이트하는 것을 도와주세요.
  • 접근성 개선: 스크린 리더 테스트, WCAG 준수, 키보드 내비게이션.
