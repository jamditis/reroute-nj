새 기능: 웹사이트에 Reroute NJ 삽입

작성자 Joe Amditis 2026년 2월 13일

Portal Bridge 전환이 토요일에 시작됩니다. New Jersey에서 교통을 다루거나 — 커뮤니티 사이트, 도시 Facebook 페이지 또는 통근자 포럼을 운영하는 경우 — 이제 Reroute NJ 도구를 웹사이트에 직접 배치할 수 있습니다.

방금 완전한 삽입 시스템을 출시했습니다. 4가지 출력 형식, 삽입 코드를 만들어주는 시각적 구성 도구, 소셜 미디어와 이메일 뉴스레터를 위한 PNG 내보내기. 모든 것이 무료입니다.

삽입하는 4가지 방법

정보 카드 — 노선 영향, 역 세부정보 또는 전체 전환 요약을 보여주는 간결한 카드입니다. 기사 사이드바나 라이브 블로그 상단에 넣으세요. 세 가지 카드 유형: 노선 카드(한 노선의 영향), 역 카드(한 역의 변경사항), 요약 카드(5개 노선 전체 개요).

인터랙티브 위젯 — 노선 안내, 비교 도구, 뉴스 피드 또는 지도의 미니 버전. 독자들이 페이지 내에서 상호작용할 수 있는 실시간 작동 도구입니다. 도구를 선택하면 맞춤형 크기의 iframe에 로드됩니다.

전체 도구 삽입 — 헤더와 푸터가 제거된 Reroute NJ 페이지. 모든 도구 URL에 ?embed=true를 추가하면 삽입 가능해집니다. 독자에게 완전한 경험을 제공하려는 전용 전환 페이지에 적합합니다.

스크립트 태그 삽입 — 하나의 <script> 태그와 하나의 <div>. iframe 구성이 필요 없습니다. 삽입 유형, 노선 및 역에 대한 데이터 속성이 있는 div를 추가하고 스크립트를 포함하면 나머지를 처리합니다. iframe 코드 붙여넣기가 제한된 CMS에서 작동합니다.

구성 도구

삽입 페이지에는 삽입 유형 선택, 노선과 역 선택, 테마(밝은/어두운) 선택, 사용자 지정 강조 색상 설정을 안내하는 시각적 구성 도구가 있습니다. 실시간 미리보기를 표시하고 4가지 형식 모두로 복사-붙여넣기 가능한 코드를 제공합니다.

정보 카드의 경우 카드를 고해상도 이미지로 렌더링하는 PNG 다운로드 버튼도 있습니다. PNG 내보내기는 Canvas 렌더링을 사용하므로(스크린샷이 아님) 텍스트가 어떤 크기에서도 선명하게 유지됩니다. 뉴스레터, 트윗 또는 인쇄물에 사용하세요.

모든 스타일과 데이터가 번들로 제공되는 독립 파일을 제공하는 HTML 다운로드 옵션도 있습니다. 외부 종속성 없음, 네트워크 요청 없음. 브라우저에서 열면 오프라인에서도 작동합니다.

사용자 지정

모든 삽입은 두 가지 사용자 지정 옵션을 지원합니다:

  • 테마 — 밝은 또는 어두운. 삽입 URL에 &theme=dark를 추가하거나 스크립트 태그 div에 data-theme="dark"를 설정하세요.
  • 강조 색상 — 모든 16진수 색상. &accent=FF6B35를 추가하거나 data-accent="FF6B35"를 설정하세요. 강조색은 CTA 버튼과 색상 강조에 적용됩니다.

코드를 건드리지 않고도 사이트 디자인에 맞게 삽입을 조정하세요.

뉴스룸을 위한 안내

우리는 이것을 여러분을 위해 만들었습니다. 전환은 한 달 동안 지역의 교통 보도를 지배할 것이며, 독자들은 기사뿐만 아니라 도구가 필요합니다. 사이드바의 노선 카드는 Montclair-Boonton 통근자에게 정확히 무엇이 변경되는지 알려줍니다. 비교 위젯을 통해 독자는 자신의 역을 입력하고 모든 대체 경로를 볼 수 있습니다.

삽입 코드는 WordPress, Squarespace, Ghost, Substack(스크립트 태그 형식 사용), HTML을 허용하는 모든 플랫폼에서 작동합니다. 특정 플랫폼에 삽입하는 데 문제가 있으면 이슈를 열거나 amditisj@montclair.edu로 이메일을 보내시면 도와드리겠습니다.

모든 것이 MIT 라이선스입니다. 출처 표시가 필요하지 않지만 reroutenj.org로 링크를 걸어주시면 감사하겠습니다.

다른 새로운 기능

삽입 시스템과 함께 이번 릴리스에는 다음이 포함됩니다:

  • 11개 언어로 된 블로그 — 블로그 게시물이 사이트의 나머지 부분과 함께 번역되었습니다. 블로그에는 적절한 게시물 구조를 갖춘 자체 인덱스 페이지가 있습니다.
  • SEO 및 AI 검색 최적화 — 구조화된 데이터(JSON-LD), 전체 hreflang 교차 참조, AI 봇 안내가 포함된 robots.txt, AI 검색 엔진을 위한 llms.txt, 79개 페이지를 모두 포함하는 sitemap.xml.
  • 독립형 HTML 다운로드 — 인터넷 연결 없이 작동하는 독립형 HTML 파일로 정보 카드를 내보냅니다.
