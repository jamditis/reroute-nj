Para redações e editoras

O Reroute NJ foi projetado para jornalistas, redações e editoras comunitárias. Veja como você pode usá-lo.

Incorpore nos seus artigos

  • Use o gerador de código de incorporação acima para obter um trecho iframe.
  • Cole no seu CMS (WordPress, Ghost, Squarespace, etc.) — a maioria aceita blocos de HTML.
  • A incorporação é responsiva e funciona em dispositivos móveis.
  • Não precisa de chave de API ou conta. Simplesmente funciona.

Linke a partir da sua cobertura

  • Adicione um link "Planeje seu trajeto" nas suas matérias sobre o Portal Bridge.
  • Linke para ferramentas específicas com base no assunto do artigo.
  • Já temos imagens OG e cards para redes sociais configurados para pré-visualizações.
  • Texto de âncora sugerido: "Use a ferramenta gratuita Reroute NJ para planejar seu trajeto."

Contribua com cobertura

  • Nós selecionamos a cobertura do Portal Bridge na nossa página de notícias.
  • Para sugerir um artigo, abra uma issue no GitHub com a URL e detalhes.
  • Ou envie um email para amditisj@montclair.edu com "Reroute NJ coverage" no assunto.
  • Incluímos toda cobertura relevante — opinião, análise, reportagem comunitária e mais.

Co-branding e personalização

Quer rodar o Reroute NJ com a sua própria marca? Como é de código aberto, você pode.

Opção 1: Faça um fork do projeto

1

Faça um fork do repositório no GitHub para a conta da sua organização.

2

Edite css/styles.css para alterar cores, fontes e identidade visual. Todos os valores do tema são propriedades CSS customizadas no bloco :root.

3

Substitua o nome da marca, logo e créditos no rodapé pelos seus. Adicione o cabeçalho/navegação da sua redação se desejar.

4

Faça o deploy no GitHub Pages, Netlify, Vercel ou no seu próprio servidor. Não precisa de etapa de build — é apenas HTML estático.

Opção 2: Incorpore com sua própria introdução

1

Incorpore a ferramenta usando o código iframe acima.

2

Adicione seu próprio cabeçalho, texto introdutório e identidade visual ao redor do iframe.

3

Estilize a página ao redor para combinar com o design da sua publicação.

Como contribuir

O Reroute NJ é um projeto comunitário. Veja como você pode ajudar, independente do seu nível técnico.

Contribuições não técnicas

  • Sugira artigos: Conhece algum artigo sobre o Portal Bridge que não incluímos? Abra uma issue ou envie email para joe@amditis.com.
  • Reporte imprecisões: Se dados de horários ou tempos de transferência estiverem errados, nos avise.
  • Compartilhe a ferramenta: Conte para seus amigos que pegam trem, compartilhe nas redes sociais, poste no grupo do seu bairro.
  • Traduza: Ajude-nos a traduzir a ferramenta para espanhol ou outros idiomas falados pelos passageiros da NJ Transit.

Contribuições técnicas

  • Reporte bugs: Encontrou um bug? Abra uma issue no GitHub.
  • Envie um pull request: O código é HTML/CSS/JS puro, sem etapa de build. Faça fork, altere e envie o PR.
  • Adicione dados: Ajude-nos a manter o data/coverage.json atualizado com novos artigos.
  • Melhore a acessibilidade: Testes com leitor de tela, conformidade WCAG, navegação por teclado.
