અમે Reroute NJ શા માટે બનાવ્યું

લેખક Joe Amditis 12 ફેબ્રુઆરી, 2026

15 ફેબ્રુઆરીએ, NJ Transit તેના ઇતિહાસમાં સૌથી મોટો સેવા વિક્ષેપ અમલમાં મૂકી રહ્યો છે. ચાર અઠવાડિયા માટે, જ્યારે Amtrak નવા Portal North Bridge ને Northeast Corridor સાથે જોડે છે, Atlantic City Rail Line સિવાય દરેક રેલ લાઇન પ્રભાવિત થશે. New Jersey અને New York વચ્ચેની લગભગ અડધી ટ્રેનો કપાશે.

હજારો મુસાફરોએ ટૂંકા સમયમાં શોધવાની જરૂર છે કે તેમની દૈનિક દિનચર્યા કેવી રીતે બદલાય છે. સત્તાવાર સંસાધનો હંમેશાં સ્પષ્ટ, વ્યક્તિગત જવાબ આપતા નથી જે તમને સવારે 6:30 વાગ્યે કામ પર જવાનો પ્રયાસ કરતી વખતે જરૂર હોય.

Reroute NJ મદદ કરવાનો પ્રયાસ છે.

પાંચ ટૂલ્સ, અગિયાર ભાષાઓ

અમે પાંચ ઇન્ટરેક્ટિવ ટૂલ્સ બનાવ્યા છે જે મુસાફરોને તેમના પૂછતા પ્રશ્નોના જવાબ આપવામાં મદદ કરે છે:

લાઇન માર્ગદર્શિકા — તમારી મુસાફરી કેવી રીતે બદલાય છે, કયા વૈકલ્પિક માર્ગો ઉપલબ્ધ છે અને તમને કઈ ટિકિટ જોઈએ છે તે જોવા માટે તમારી NJ Transit લાઇન અને સ્ટેશન પસંદ કરો. બંને દિશાઓ માટે કામ કરે છે: NYC જતા સવારના મુસાફરો અને ઘરે પાછા આવતા સાંજના મુસાફરો.

મુસાફરી સરખામણી — તમારું NJ સ્ટેશન અને તમારું Manhattan ગંતવ્ય પસંદ કરો, અને વિઝ્યુઅલ સમય વિશ્લેષણ સાથે દરેક રૂટ વિકલ્પને સાથે જુઓ. PATH vs. ફેરી vs. બસ, કુલ મુસાફરી સમય દ્વારા રેન્ક કરેલ, ખર્ચ અને ટ્રાન્સફર વિગતો સાથે.

ઇન્ટરેક્ટિવ નકશો — સ્ટેશન માર્કર્સ, મુખ્ય ટ્રાન્સફર પોઇન્ટ્સ (Hoboken Terminal, Secaucus Junction, Newark Penn) અને Hackensack River પર Portal Bridge સ્થાન સાથે નકશા પર રેન્ડર કરેલી બધી પાંચ પ્રભાવિત NJ Transit લાઇનો.

એમ્બેડ અને શેર — Iframe એમ્બેડ કોડ્સ, સીધી લિંક્સ અને ન્યૂઝરૂમ અને પ્રકાશકો માટે સૂચનાઓ જે આમાંથી કોઈપણ ટૂલને તેમની પોતાની સાઇટ પર રિપબ્લિશ કરવા માંગે છે.

દરેક ટૂલ પેજ 11 ભાષાઓમાં ઉપલબ્ધ છે: અંગ્રેજી, સ્પેનિશ, ચાઇનીઝ, ટાગાલોગ, કોરિયન, પોર્ટુગીઝ, ગુજરાતી, હિન્દી, ઇટાલિયન, અરબી અને પોલિશ. આ મશીન-અનુવાદિત પછીના વિચારો નથી — દરેક ભાષાના પોતાના પેજનો સેટ અનુવાદિત નેવિગેશન, લેબલ્સ, વર્ણનો અને મેટાડેટા સાથે છે. સ્ટેશન નામો, લાઇન નામો અને સ્થળ નામો અંગ્રેજીમાં રહે છે કારણ કે તે સંકેતો પર શું છે.

તે શા માટે આ રીતે બનાવવામાં આવ્યું છે

Reroute NJ એક શૂન્ય-બિલ્ડ સ્ટેટિક સાઇટ છે. સાદા HTML, CSS અને JavaScript. કોઈ ફ્રેમવર્ક નહીં, npm નહીં, કોઈ બિલ્ડ સ્ટેપ નહીં. તમે રિપો ફોર્ક કરી શકો છો, ફાઇલ સંપાદિત કરી શકો છો અને મિનિટોમાં GitHub Pages પર તેને ડિપ્લોય કરી શકો છો.

અમે ઇચ્છતા હતા કે પ્રોજેક્ટ કોઈપણને સુલભ હોય જે યોગદાન આપવા માંગે છે — પત્રકારો જે કવરેજ ફીડમાં લેખો ઉમેરી શકે છે, સમુદાય સભ્યો જે અચોક્કસતા રિપોર્ટ કરી શકે છે, અને વિકાસકર્તાઓ જે સુવિધાઓ ઉમેરી શકે છે. યોગદાન માટેનો અવરોધ ઘટાડવો એ નવીનતમ ટેક્નોલોજીનો ઉપયોગ કરતાં વધુ મહત્વપૂર્ણ છે.

બધું ક્લાયંટ-સાઇડ છે. કોઈ સર્વર નહીં, ડેટાબેઝ નહીં, API કી નહીં. ડેટા JSON અને JavaScript ઑબ્જેક્ટ્સ તરીકે બંડલ કરવામાં આવે છે. સાઇટ ઝડપી છે અને GitHub ના સર્વર્સ લોડ હેઠળ હોય તો પણ કામ કરે છે.

ન્યૂઝરૂમ અને પ્રકાશકો માટે

અમે Reroute NJ ને શેર કરવા, એમ્બેડ કરવા અને રિપબ્લિશ કરવા માટે બનાવી છે. જો તમે સ્થાનિક સમાચાર સાઇટ, સમુદાય બ્લોગ અથવા ટ્રાન્ઝિટ એડવોકસી પેજ ચલાવો છો, તો અમે ઇચ્છીએ છીએ કે તમે આ ટૂલ્સનો ઉપયોગ કરો.

  • કોઈપણ ટૂલ એમ્બેડ કરો તમારી વેબસાઇટ પર અમારા એમ્બેડ કોડ જનરેટરનો ઉપયોગ કરીને. Iframe કૉપિ અને પેસ્ટ કરો.
  • ટૂલ્સને લિંક કરો તમારા Portal Bridge કવરેજમાંથી. URL કાયમી છે અને સ્વચ્છ પૂર્વાવલોકન માટે સોશિયલ મેટા ટેગ્સ ધરાવે છે.
  • પ્રોજેક્ટ ફોર્ક કરો અને તમારું પોતાનું બ્રાન્ડેડ સંસ્કરણ ચલાવો. કોડ MIT-લાઇસન્સવાળો છે. રંગો બદલો, તમારો લોગો ઉમેરો, તમારા પોતાના ડોમેન પર ડિપ્લોય કરો.
  • તમારું કવરેજ ઉમેરો અમારા સમાચાર ફીડ પર. તમારી લેખ લિંક્સ સાથે GitHub issue ખોલો અથવા amditisj@montclair.edu પર ઇમેઇલ કરો.

અમને સ્થાનિક પ્રકાશકો અને સામુદાયિક સમાચાર આઉટલેટ્સ પાસેથી સાંભળવામાં રસ છે જે કટઓવરથી પ્રભાવિત ચોક્કસ શહેરોને સેવા આપે છે. તમારું સ્થાનિક કવરેજ મુસાફરોને જોઈએ છે, અને અમે તેને તેમની સામે મેળવવામાં મદદ કરવા માંગીએ છીએ.

તમે કેવી રીતે મદદ કરી શકો

આ એક સામુદાયિક પ્રોજેક્ટ છે અને અમને સમુદાયની મદદની જરૂર છે:

  • ટૂલ શેર કરો. તમારા ટ્રેન મિત્રોને ટેક્સ્ટ કરો. તમારા શહેરના Facebook ગ્રૂપમાં પોસ્ટ કરો. વધુ લોકો તેના વિશે જાણે છે, તેટલું સારું તે કામ કરે છે.
  • અચોક્કસતા રિપોર્ટ કરો. જો મુસાફરી સમય અંદાજ ખોટો છે, અથવા કોઈ સમયપત્રક ફેરફાર પ્રતિબિંબિત નથી, તો અમને જણાવો.
  • લેખો સૂચવો. અમે સૂચિબદ્ધ કર્યા નથી તેવું સારું Portal Bridge કવરેજ મળ્યું? સબમિટ કરો.
  • કોડ યોગદાન આપો. GitHub રિપો ખુલ્લી છે. કોઈ બિલ્ડ સ્ટેપ નથી, કોઈ ફ્રેમવર્ક જ્ઞાન જરૂરી નથી. જો તમે HTML સંપાદિત કરી શકો છો, તો તમે યોગદાન આપી શકો છો.
  • એક્સેસિબિલિટીમાં મદદ કરો. અમે ઇચ્છીએ છીએ કે ટૂલ દરેક માટે કામ કરે. સ્ક્રીન રીડર ટેસ્ટિંગ, કીબોર્ડ નેવિગેશન સુધારાઓ અને WCAG અનુપાલન સમીક્ષા બધા સ્વાગત છે.

મોટું ચિત્ર

Portal Bridge કટઓવર પીડાદાયક છે, પરંતુ તે સારી વસ્તુ છે. 115 વર્ષ જૂનો Portal Bridge દાયકાઓથી Northeast Corridor પરનો સૌથી ખરાબ અવરોધ રહ્યો છે. જ્યારે પણ તે બોટ ટ્રાફિક માટે ખોલતી વખતે અટકી જાય છે, સાંકળ વિલંબ હજારો મુસાફરોને અસર કરે છે. નવો Portal North Bridge તે સમસ્યાને દૂર કરે છે.

આ ચાર અઠવાડિયાનો વિક્ષેપ પ્રગતિની કિંમત છે. જ્યારે ફેઝ 2 2026 ના પાનખરમાં આવે છે, અમે તે પણ આવરી લેવા માટે Reroute NJ અપડેટ કરીશું.

દરમિયાન, અમે આશા રાખીએ છીએ કે આ ટૂલ્સ આગામી મહિનાને નેવિગેટ કરવા માટે થોડું સરળ બનાવે છે. આગળ આયોજન કરો, પ્લેટફોર્મ પર એકબીજા સાથે ધીરજ રાખો, અને યાદ રાખો: આ કામચલાઉ છે.
