અમારી પદ્ધતિ વિશે

Reroute NJ મુસાફરોને ચોક્કસ માર્ગદર્શન આપે છે: કયું ટ્રેન લેવું, કઈ ટિકિટ ખરીદવી, ક્યાં ટ્રાન્સફર કરવું. આ માહિતી સાચી હોવી જોઈએ. લોકો સમયસર કામ પર પહોંચવા માટે તેના પર આધાર રાખે છે. અમારો અભિગમ બીજા બધા કરતાં ચોકસાઈ અને ઍક્સેસિબિલિટીને પ્રાથમિકતા આપે છે.

અમે અનુવાદો કેવી રીતે બનાવીએ છીએ

આ સાઇટ પરનું દરેક ટૂલ 11 ભાષાઓમાં ઉપલબ્ધ છે: અંગ્રેજી, સ્પેનિશ, ચાઇનીઝ, ટાગાલોગ, કોરિયન, પોર્ટુગીઝ, ગુજરાતી, હિંદી, ઇટાલિયન, અરબી અને પોલિશ. આ ભાષાઓ પસંદ કરવામાં આવી કારણ કે U.S. Census ડેટા અનુસાર આ New Jersey માં સૌથી વધુ બોલાતી ભાષાઓ છે.

અનુવાદો AI ભાષા મોડેલ્સની મદદથી બનાવવામાં આવે છે અને કુદરતી ભાષા અને ચોકસાઈ માટે સમીક્ષા કરવામાં આવે છે. દરેક ભાષાને અનુવાદિત નેવિગેશન, લેબલ્સ, હેડિંગ્સ, વર્ણનો, ઍક્સેસિબિલિટી ટેક્સ્ટ અને મેટાડેટા સાથે HTML પેજનો પોતાનો સંપૂર્ણ સેટ મળે છે. આ બ્રાઉઝર ઑટો-ટ્રાન્સલેટ ઓવરલે નથી — દરેક અનુવાદિત પેજ એક સ્ટેન્ડઅલોન દસ્તાવેજ છે જે જરૂર પડ્યે JavaScript વિના કામ કરે છે.

અમે ચોક્કસ નિયમો અનુસરીએ છીએ કે શું અનુવાદ થાય અને શું ન થાય:

  • સ્ટેશનના નામ અંગ્રેજીમાં રહે છે. "Secaucus Junction" એ પ્લેટફોર્મ પરના સાઇન પર જે લખ્યું છે તે છે. તેનો અનુવાદ કરવાથી તમારું ટ્રેન શોધવું મુશ્કેલ થશે, સરળ નહીં.
  • લાઇનના નામ અંગ્રેજીમાં રહે છે. "Northeast Corridor" અને "Montclair-Boonton Line" NJ Transit ના અધિકૃત નામો છે. તે દરેક શેડ્યૂલ, એપ અને સાઇન પર અંગ્રેજીમાં દેખાય છે.
  • સ્થળોના નામ અંગ્રેજીમાં રહે છે. "Hoboken Terminal," "Penn Station New York," અને "Port Authority" યોગ્ય સંજ્ઞાઓ છે. મુસાફરોએ જે જુએ છે તેને જે વાંચે છે તેની સાથે મેળવવાની જરૂર છે.
  • બાકી બધું અનુવાદ થાય છે. સૂચનાઓ, વર્ણનો, લેબલ્સ, નેવિગેશન, બટન ટેક્સ્ટ, ઍક્સેસિબિલિટી જાહેરાતો, ભૂલ સંદેશાઓ અને પેજ મેટાડેટા — બધું દરેક ભાષામાં અનુવાદ થાય છે.

આ એક જાણીજોઈને લીધેલો નિર્ણય છે: મુસાફરો સ્ટેશન પર ખરેખર જે જુએ છે અને સાંભળે છે તેની સાથે વફાદારી ભાષાકીય સુસંગતતા કરતાં વધુ મહત્ત્વની છે. ગુજરાતી બોલતા મુસાફરને તેમના ફોન પર "Hoboken Terminal માટે ટ્રેન લો" વાંચવાની જરૂર છે — "હોબોકન ટર્મિનલ" નહીં — કારણ કે પ્લેટફોર્મ ઉપરના સાઇન પર "Hoboken Terminal" લખ્યું છે.

અમે ટ્રાન્ઝિટ ડેટા કેવી રીતે ચકાસીએ છીએ

આ સાઇટ પરનો દરેક દાવો — ટ્રેનની સંખ્યા, શેડ્યૂલ ફેરફારો, ભાડાની માહિતી, ટ્રાન્સફર દિશાઓ — અધિકૃત સ્ત્રોત સુધી ટ્રેસ કરી શકાય છે. અમે સાઇટેશન ડેટાબેસ જાળવીએ છીએ જે 28 ચોક્કસ દાવાઓને અધિકૃત NJ Transit, Amtrak, PATH અને NY Waterway પેજ સાથે જોડે છે.

અમારી ચકાસણી પ્રક્રિયા:

  • પહેલાં અધિકૃત સ્ત્રોતો. ટ્રેનની સંખ્યા, ડાયવર્ઝન, ક્રોસ-ઓનરિંગ નીતિઓ અને ભાડાની માહિતી njtransit.com/portalcutover અને NJ Transit ની અધિકૃત પ્રેસ રિલીઝમાંથી આવે છે. અમે ટ્રાન્ઝિટ ડેટા માટે પ્રાથમિક સ્ત્રોત તરીકે સેકન્ડહેન્ડ રિપોર્ટિંગનો ઉપયોગ કરતા નથી.
  • ઓટોમેટેડ ચકાસણી. અમે 14 ટેસ્ટ સ્યુટ્સમાં 698+ ઓટોમેટેડ ચકાસણીઓ ચલાવીએ છીએ જે ડેટા સ્ટ્રક્ચર, ટ્રાન્ઝિટ ફેક્ટ્સ, પેજ વચ્ચે ક્રોસ-રેફરન્સ, HTML ઇન્ટેગ્રિટી, અનુવાદ સંપૂર્ણતા અને ઍક્સેસિબિલિટી અનુપાલન ચકાસે છે. આ ટેસ્ટ દરેક ફેરફાર પ્રકાશિત થાય તે પહેલાં ચાલે છે.
  • ઓટોમેટેડ મોનિટરિંગ. એક સ્ક્રેપર દિવસમાં ચાર વખત NJ Transit અને Amtrak ના અધિકૃત પેજ શેડ્યૂલ અથવા નીતિ ફેરફારો માટે ચકાસે છે. જ્યારે સામગ્રી બદલાય, ત્યારે અમે સમીક્ષા કરીએ છીએ અને સાઇટ અપડેટ કરીએ છીએ.
  • દરેક પેજ પર સ્ત્રોત ઍટ્રિબ્યુશન. દરેક લાઇન ગાઇડ કાર્ડ અને મુસાફરી સરખામણી પરિણામમાં તેના દાવાઓનું સમર્થન કરતા અધિકૃત સ્ત્રોતોની લિંક્સ શામેલ છે, જેથી મુસાફરો પોતે ચકાસી શકે.
  • ઓપન સોર્સ. સંપૂર્ણ કોડબેસ, તમામ ડેટા ફાઇલ્સ અને તમામ ટેસ્ટ સ્યુટ્સ GitHub પર જાહેર છે. કોઈ પણ ડેટાની તપાસ, પડકાર અથવા સુધારો કરી શકે છે.

અમારા ઍક્સેસિબિલિટી ધોરણો

Accessibility shaped every design decision from the start. The site meets WCAG 2.1 AA, the international standard for web accessibility.

વ્યવહારમાં તેનો અર્થ શું છે:

  • ઉચ્ચ કોન્ટ્રાસ્ટ મોડ. દરેક પેજ પર એક ટૉગલ ઓછી દૃષ્ટિ ધરાવતા મુસાફરો માટે ઉચ્ચ-કોન્ટ્રાસ્ટ કલર સ્કીમ પર સ્વિચ કરે છે. સેટિંગ પેજ વચ્ચે ટકે છે.
  • સરળ દૃશ્ય. ઓછી વિક્ષેપની જરૂરિયાત ધરાવતા મુસાફરો માટે વિઝ્યુઅલ નોઇઝ ઘટાડવા સુશોભન તત્વો દૂર કરે છે.
  • કીબોર્ડ નેવિગેશન. દરેક કંટ્રોલ કીબોર્ડ દ્વારા પહોંચી શકાય છે. ફોકસ ઇન્ડિકેટર્સ દેખાય છે. ટેબ ક્રમ વિઝ્યુઅલ લેઆઉટને અનુસરે છે.
  • સ્ક્રીન રીડર સપોર્ટ. ARIA લેબલ્સ, ભૂમિકાઓ અને લાઇવ રિજન્સ સ્ક્રીન રીડર્સને સામગ્રી ફેરફારોની જાહેરાત કરવા દે છે. દરેક પેજ પર સ્કિપ-ટુ-કન્ટેન્ટ લિંક્સ.
  • ટચ ટાર્ગેટ્સ. તમામ બટન્સ અને કંટ્રોલ્સ મોબાઇલ પર ઓછામાં ઓછા 44 પિક્સેલ છે — એક હલતા ટ્રેનમાં ચોક્કસ રીતે ટેપ કરવા માટે પૂરતા મોટા.
  • જમણેથી ડાબે સપોર્ટ. અરબી અનુવાદ યોગ્ય જમણેથી ડાબે લેઆઉટ, ટેક્સ્ટ દિશા અને મિરર કરેલી નેવિગેશન સાથે રેન્ડર થાય છે.
  • પ્રિન્ટ સ્ટાઇલશીટ્સ. જે મુસાફરો કાગળની બેકઅપ ઇચ્છે છે તેમના માટે દરેક પેજ સ્વચ્છ રીતે પ્રિન્ટ થાય છે.

અમે આ રીતે કેમ બનાવ્યું

Reroute NJ is a plain HTML, CSS, and JavaScript site. No frameworks, no build step, no server, no database. We chose this stack for reliability.

કટઓવર દરમિયાન, સેંકડો હજારો મુસાફરોને પીક કલાકોમાં આ માહિતીની જરૂર પડશે. CDN થી સર્વ થતી સ્ટેટિક સાઇટ ટ્રાફિક સ્પાઇક્સ તૂટ્યા વિના સંભાળે છે. તે જૂના ફોન, ધીમા કનેક્શન અને દરેક બ્રાઉઝર પર કામ કરે છે. ડાઉન થવા માટે કોઈ સર્વર નથી, રેટ-લિમિટ કરવા માટે કોઈ API નથી, તૂટવા માટે કોઈ ડિપેન્ડન્સી નથી.

ટૂલ્સ એ વ્યક્તિ માટે ડિઝાઇન કરવામાં આવ્યા છે જે સવારે 6:30 વાગ્યે ફોન ચેક કરે છે, પ્લેટફોર્મ પર ઊભો છે, અને તેનું ટ્રેન હજી ચાલે છે કે નહીં તે જાણવાનો પ્રયત્ન કરે છે. તે વ્યક્તિને ઝડપથી સ્પષ્ટ જવાબની જરૂર છે. તેમને એનિમેશન, લોડિંગ સ્પિનર્સ અથવા સાઇનઅપ ફોર્મની જરૂર નથી. દરેક ડિઝાઇન નિર્ણય તે પરિસ્થિતિથી શરૂ થાય છે.

અમે શું ખોટું કરીએ છીએ

અમે ભૂલો કરીએ છીએ. જ્યારે અમે તે શોધીએ છીએ, ત્યારે અમે તે સુધારીએ છીએ અને ફેરફાર દસ્તાવેજ કરીએ છીએ. કેટલીક બાબતો જાણવા જેવી:

  • મુસાફરી સમયના અંદાજ અંદાજિત છે. અમે પ્રકાશિત શેડ્યૂલ અને સરેરાશ મુસાફરી સમયનો ઉપયોગ કરીએ છીએ, લાઇવ ડેટાનો નહીં. ખરેખરનો સમય બદલાશે, ખાસ કરીને કટઓવર દરમિયાન જ્યારે સિસ્ટમ દબાણ હેઠળ હોય.
  • શેડ્યૂલ બદલાય છે. NJ Transit કટઓવર દરમિયાન રાઇડરશિપ પેટર્ન પર આધારિત સર્વિસ એડજસ્ટ કરી શકે છે. અમે દિવસમાં ચાર વખત ફેરફારો માટે મોનિટર કરીએ છીએ, પરંતુ અધિકૃત અપડેટ અને અમારી સાઇટ તેને પ્રતિબિંબિત કરે છે તે વચ્ચે વિલંબ હોઈ શકે છે.
  • અનુવાદોમાં ભૂલો હોઈ શકે છે. જ્યારે અમે અનુવાદો કુદરતી અને ચોક્કસ બનાવવા માટે કામ કરીએ છીએ, ત્યારે અમે AI-સહાયિત અનુવાદ પર આધાર રાખીએ છીએ. જો તમને કોઈ ભાષામાં ભૂલ મળે, તો કૃપા કરીને તેની જાણ કરો અને અમે તે સુધારીશું.

આ એક સ્વતંત્ર પ્રોજેક્ટ છે. અમે NJ Transit, Amtrak અથવા કોઈ સરકારી એજન્સી સાથે સંબંધિત નથી. મુસાફરી પહેલાં હંમેશાં njtransit.com પર મહત્ત્વની મુસાફરી માહિતી ચકાસો.
