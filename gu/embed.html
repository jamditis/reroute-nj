લાઇન માર્ગદર્શિકા મુસાફરી સરખામણી સમાચાર નકશો એમ્બેડ અને શેર

તમારી વેબસાઇટ પર Reroute NJ મૂકો

Reroute NJ મફત, ઓપન સોર્સ અને સમુદાય માટે બનાવેલ છે. સમાચાર રૂમ, પ્રકાશકો, સામુદાયિક સંસ્થાઓ અને કોઈપણ અમારા સાધનોને એમ્બેડ, લિંક અથવા રિપબ્લિશ કરી શકે છે.

એમ્બેડ કોડ જનરેટર

Reroute NJ ના કોઈપણ સાધનને તમારી વેબસાઇટ પર મૂકવા iframe એમ્બેડ કોડ કૉપિ અને પેસ્ટ કરો. સાધન iframe ની અંદર સંપૂર્ણ ઇન્ટરેક્ટિવ અને રિસ્પોન્સિવ છે.

કૉપિ થયું!

સીધી લિંક્સ

કોઈપણ સાધનને સીધી લિંક કરો. આ URL કાયમી અને શેર કરી શકાય તેવા છે.

સમાચાર રૂમ અને પ્રકાશકો માટે

Reroute NJ પત્રકારો, સમાચાર રૂમ અને સામુદાયિક પ્રકાશકો દ્વારા ઉપયોગ માટે બનાવેલ છે. તમે તેનો ઉપયોગ કેવી રીતે કરી શકો તે અહીં છે.

તમારા લેખોમાં એમ્બેડ કરો

  • iframe સ્નિપેટ મેળવવા ઉપરનો એમ્બેડ કોડ જનરેટર વાપરો.
  • તમારા CMS (WordPress, Ghost, Squarespace, વગેરે) માં પેસ્ટ કરો — મોટાભાગના રૉ HTML બ્લૉક્સ સપોર્ટ કરે છે.
  • એમ્બેડ રિસ્પોન્સિવ છે અને મોબાઇલ પર કામ કરે છે.
  • API કી અથવા એકાઉન્ટની જરૂર નથી. તે સીધું જ કામ કરે છે.

તમારા કવરેજમાંથી લિંક કરો

  • તમારી Portal Bridge વાર્તાઓમાં "તમારી મુસાફરી પ્લાન કરો" લિંક બૉક્સ ઉમેરો.
  • લેખ શું આવરી લે છે તેના આધારે ચોક્કસ સાધનોને લિંક કરો.
  • સોશિયલ મીડિયા પ્રિવ્યુ માટે OG ઇમેજ અને સોશિયલ કાર્ડ પહેલેથી સેટ છે.
  • સૂચવેલ એન્કર ટેક્સ્ટ: "તમારી મુસાફરી પ્લાન કરવા મફત Reroute NJ સાધન વાપરો."

કવરેજ ઉમેરો

  • અમે સમાચાર કવરેજ પેજ પર Portal Bridge કવરેજ સંગ્રહિત કરીએ છીએ.
  • લેખ સૂચવવા માટે, URL અને વિગતો સાથે GitHub ઇશ્યૂ ખોલો.
  • અથવા "Reroute NJ coverage" વિષય લાઇનમાં amditisj@montclair.edu પર ઇમેઇલ કરો.
  • અમે બધા સંબંધિત કવરેજ સામેલ કરીએ છીએ — અભિપ્રાય, વિશ્લેષણ, સમુદાય રિપોર્ટિંગ, અને વધુ.

કો-બ્રાન્ડિંગ અને કસ્ટમાઇઝેશન

તમારી પોતાની બ્રાન્ડિંગ સાથે Reroute NJ ચલાવવા માંગો છો? ઓપન સોર્સ હોવાથી, તમે કરી શકો.

વિકલ્પ 1: પ્રોજેક્ટ ફોર્ક કરો

1

તમારી સંસ્થાના એકાઉન્ટ પર GitHub રિપોઝિટરી ફોર્ક કરો.

2

રંગો, ફોન્ટ અને બ્રાન્ડિંગ બદલવા css/styles.css એડિટ કરો. બધા થીમ મૂલ્યો :root બ્લૉકમાં CSS કસ્ટમ પ્રોપર્ટીઝ છે.

3

બ્રાન્ડ નામ, લોગો અને ફૂટર ક્રેડિટ તમારા પોતાના સાથે બદલો. જો ઈચ્છો તો તમારા ન્યૂઝરૂમનું હેડર/નેવ ઉમેરો.

4

GitHub Pages, Netlify, Vercel અથવા તમારા પોતાના સર્વર પર ડિપ્લોય કરો. કોઈ બિલ્ડ સ્ટેપ જરૂરી નથી — તે ફક્ત સ્ટેટિક HTML છે.

વિકલ્પ 2: તમારા પોતાના પરિચય સાથે એમ્બેડ કરો

1

ઉપરના iframe કોડનો ઉપયોગ કરીને સાધન એમ્બેડ કરો.

2

iframe ની આસપાસ તમારું પોતાનું હેડર, પરિચય ટેક્સ્ટ અને બ્રાન્ડિંગ ઉમેરો.

3

આસપાસના પેજને તમારા પ્રકાશનની ડિઝાઇન સાથે મેચ કરવા સ્ટાઇલ કરો.

કેવી રીતે યોગદાન આપવું

Reroute NJ એક સામુદાયિક પ્રોજેક્ટ છે. તમારા ટેકનિકલ કૌશલ્ય સ્તરને ધ્યાનમાં લીધા વિના, તમે કેવી રીતે મદદ કરી શકો તે અહીં છે.

બિન-ટેકનિકલ યોગદાન

  • લેખ સૂચનો સબમિટ કરો: અમે ચૂકી ગયેલો Portal Bridge લેખ જાણો છો? ઇશ્યૂ ખોલો અથવા joe@amditis.com પર ઇમેઇલ કરો.
  • ખોટી માહિતી રિપોર્ટ કરો: જો સમયપત્રક ડેટા અથવા ટ્રાન્સફર સમય ખોટો હોય, તો અમને જણાવો.
  • સાધન શેર કરો: તમારા મુસાફર મિત્રોને કહો, સોશિયલ મીડિયા પર શેર કરો, તમારા પડોશી જૂથમાં પોસ્ટ કરો.
  • અનુવાદ કરો: NJ Transit મુસાફરો દ્વારા બોલાતી સ્પેનિશ અથવા અન્ય ભાષાઓમાં સાધનનો અનુવાદ કરવામાં અમને મદદ કરો.

ટેકનિકલ યોગદાન

  • બગ રિપોર્ટ કરો: બગ મળ્યો? GitHub પર ઇશ્યૂ ખોલો.
  • પુલ રિક્વેસ્ટ સબમિટ કરો: કોડબેઝ વેનિલા HTML/CSS/JS છે, કોઈ બિલ્ડ સ્ટેપ નથી. ફોર્ક કરો, બદલો, PR કરો.
  • ડેટા ઉમેરો: નવા લેખો સાથે data/coverage.json અપડેટ રાખવામાં મદદ કરો.
  • એક્સેસિબિલિટી સુધારો: સ્ક્રીન રીડર ટેસ્ટિંગ, WCAG અનુપાલન, કીબોર્ડ નેવિગેશન.
