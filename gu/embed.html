એમ્બેડ અને શેર
લાઇન માર્ગદર્શિકા મુસાફરી સરખામણી સમાચાર નકશો એમ્બેડ અને શેર બ્લોગ

તમારી વેબસાઇટ પર Reroute NJ ટૂલ્સ મફત એમ્બેડ કરો. ન્યૂઝરૂમ, પ્રકાશકો અને સામુદાયિક સંસ્થાઓ કોઈપણ ટૂલ iframe કરી શકે છે, સીધું લિંક કરી શકે છે, અથવા બ્રાન્ડેડ વર્ઝન બનાવવા ઓપન-સોર્સ કોડ ફોર્ક કરી શકે છે.

તમારી વેબસાઇટ પર Reroute NJ મૂકો

સમાચાર રૂમ અને પ્રકાશકો માટે

Reroute NJ પત્રકારો, ન્યૂઝરૂમ અને સામુદાયિક પ્રકાશકો માટે ડિઝાઇન કરાયેલ છે.

તમારા લેખોમાં એમ્બેડ કરો

  • કોઈપણ ફોર્મેટ માટે એમ્બેડ કોડ જનરેટ કરવા ઉપરના કૉન્ફિગ્યુરેટરનો ઉપયોગ કરો.
  • તમારા CMS (WordPress, Ghost, Squarespace, વગેરે) માં પેસ્ટ કરો — મોટાભાગના raw HTML બ્લોક્સ સપોર્ટ કરે છે.
  • એમ્બેડ્સ રિસ્પોન્સિવ છે અને મોબાઇલ પર કામ કરે છે.
  • API કી અથવા એકાઉન્ટની જરૂર નથી.

તમારા કવરેજમાંથી લિંક કરો

  • તમારી Portal Bridge વાર્તાઓમાં "તમારી મુસાફરી પ્લાન કરો" લિંક બૉક્સ ઉમેરો.
  • લેખ શું આવરી લે છે તેના આધારે ચોક્કસ સાધનોને લિંક કરો.
  • OG ઇમેજ અને સોશિયલ કાર્ડ્સ સ્વચ્છ પ્રીવ્યૂ માટે પહેલેથી સેટ અપ છે.
  • સૂચવેલ એન્કર ટેક્સ્ટ: "તમારી મુસાફરી પ્લાન કરવા મફત Reroute NJ સાધન વાપરો."

કવરેજ ઉમેરો

  • અમે સમાચાર કવરેજ પેજ પર Portal Bridge કવરેજ સંગ્રહિત કરીએ છીએ.
  • લેખ સૂચવવા, GitHub પર issue ખોલો.
  • અથવા amditisj@montclair.edu પર "Reroute NJ coverage" વિષય સાથે ઇમેઇલ કરો.
  • અમે બધા સંબંધિત કવરેજ સામેલ કરીએ છીએ — અભિપ્રાય, વિશ્લેષણ, સમુદાય રિપોર્ટિંગ, અને વધુ.

કો-બ્રાન્ડિંગ અને કસ્ટમાઇઝેશન

બિન-ટેકનિકલ યોગદાન

  • લેખ સૂચનો સબમિટ કરો: અમે ચૂકી ગયેલો Portal Bridge લેખ જાણો છો? ઇશ્યૂ ખોલો અથવા joe@amditis.com પર ઇમેઇલ કરો.
  • ખોટી માહિતી રિપોર્ટ કરો: જો સમયપત્રક ડેટા અથવા ટ્રાન્સફર સમય ખોટો હોય, તો અમને જણાવો.
  • સાધન શેર કરો: તમારા મુસાફર મિત્રોને કહો, સોશિયલ મીડિયા પર શેર કરો, તમારા પડોશી જૂથમાં પોસ્ટ કરો.

ટેકનિકલ યોગદાન

  • બગ રિપોર્ટ કરો: બગ મળ્યો? GitHub પર ઇશ્યૂ ખોલો.
