新功能：在您的网站上嵌入 Reroute NJ

作者 Joe Amditis 2026年2月13日

Portal Bridge 切换将于周六开始。如果您报道 New Jersey 的交通 — 或者如果您运营社区网站、城镇 Facebook 页面或通勤论坛 — 您现在可以将 Reroute NJ 工具直接放在您的网站上。

我们刚刚推出了完整的嵌入系统。四种输出格式、一个为您构建嵌入代码的可视化配置器，以及用于社交媒体和电子邮件通讯的 PNG 导出。所有功能免费使用。

四种嵌入方式

信息卡片 — 显示线路影响、车站详情或完整切换摘要的紧凑卡片。将其放入文章侧边栏或实时博客顶部。三种卡片类型：线路卡片（一条线路的影响）、车站卡片（一个车站的变化）和摘要卡片（一目了然的全部五条线路）。

交互式小部件 — 线路指南、比较工具、新闻源或地图的迷你版本。这些是读者可以在您的页面内进行交互的实时工作工具。选择一个工具，它会在一个适合大小的 iframe 中加载。

完整工具嵌入 — 任何去除页眉和页脚的 Reroute NJ 页面。在任何工具 URL 后附加 ?embed=true，它就变得可以嵌入。适合专用的切换页面，在那里您希望为读者提供完整的体验。

脚本标签嵌入 — 一个 <script> 标签和一个 <div>。不需要 iframe 配置。添加带有嵌入类型、线路和车站数据属性的 div，包含脚本，它会处理其余部分。适用于粘贴 iframe 代码受限的 CMS。

配置器

嵌入页面有一个可视化配置器，它会引导您选择嵌入类型、选择线路和车站、选择主题（浅色或深色）并设置自定义强调色。它显示实时预览并为您提供所有四种格式的复制粘贴代码。

对于信息卡片，您还可以获得一个 Download PNG 按钮，它将卡片渲染为高保真图像。PNG 导出使用 Canvas 渲染（而不是截图），因此文本在任何大小下都保持清晰。在新闻通讯、tweets 或打印材料中使用它。

还有一个 Download HTML 选项，它为您提供一个包含所有样式和数据的自包含文件。没有外部依赖，没有网络请求。在浏览器中打开它即可离线工作。

自定义

每个嵌入都支持两个自定义选项：

  • Theme — 浅色或深色。在任何嵌入 URL 后附加 &theme=dark 或在脚本标签 div 上设置 data-theme="dark"。
  • Accent color — 任何十六进制颜色。附加 &accent=FF6B35 或设置 data-accent="FF6B35"。强调色应用于 CTA 按钮和颜色高亮。

将嵌入与您网站的设计相匹配，无需触及任何代码。

为新闻编辑室

我们为您构建了这个。切换将在该地区主导交通报道一个月，您的读者需要工具，而不仅仅是文章。侧边栏中的线路卡片可以准确地告诉 Montclair-Boonton 通勤者对他们有什么变化。比较小部件让读者输入他们的车站并查看每条替代路线。

嵌入代码可在 WordPress、Squarespace、Ghost、Substack（使用脚本标签格式）以及任何接受 HTML 的平台上工作。如果您在特定平台上嵌入时遇到问题，开一个 issue 或发送电子邮件至 amditisj@montclair.edu，我们会提供帮助。

所有内容均采用 MIT 许可。无需署名，但我们感谢链接回 reroutenj.org。

还有什么新功能

与嵌入系统一起，此版本还包括：

  • 11 种语言的博客 — 博客文章现在与网站的其余部分一起翻译。博客有自己的索引页面和适当的文章结构。
  • SEO 和 AI 搜索优化 — 结构化数据（JSON-LD）、完整的 hreflang 交叉引用、带有 AI 机器人指导的 robots.txt、用于 AI 搜索引擎的 llms.txt 以及覆盖所有 79 个页面的 sitemap.xml。
  • 自包含 HTML 下载 — 将任何信息卡片导出为可在没有互联网连接的情况下工作的独立 HTML 文件。
