# Reroute NJ

Free tools to help NJ Transit riders navigate the Portal North Bridge cutover (Feb 15 – Mar 15, 2026).

**Live site:** [jamditis.github.io/reroute-nj](https://jamditis.github.io/reroute-nj)

## What's here

- **Line guide** — Select your NJ Transit line and station to see how the cutover affects your commute, get route alternatives, and figure out what ticket to buy
- **Commute comparison** — Pick your station and Manhattan destination, see every route option side by side with visual time breakdowns

## Lines covered

All NJ Transit lines affected by the Portal North Bridge cutover:

- Montclair-Boonton (diverted to Hoboken)
- Morris & Essex / Gladstone Branch (diverted to Hoboken)
- Northeast Corridor (reduced service)
- North Jersey Coast Line (reduced service)
- Raritan Valley Line (Newark termination)

## Running locally

No build step. Open `index.html` in a browser or serve with any static file server:

```bash
python3 -m http.server 8000
```
