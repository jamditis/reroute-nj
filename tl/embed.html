Para sa mga newsroom at publisher

Ang Reroute NJ ay dinisenyo para gamitin ng mga mamamahayag, newsroom, at community publisher. Narito kung paano mo ito magagamit.

I-embed sa iyong mga artikulo

  • Gamitin ang embed code generator sa itaas para makuha ang iframe snippet.
  • I-paste sa iyong CMS (WordPress, Ghost, Squarespace, atbp.) — karamihan ay sumusuporta ng raw HTML block.
  • Responsive ang embed at gumagana sa mobile.
  • Hindi kailangan ng API key o account. Gumagana lang ito.

Mag-link mula sa iyong balita

  • Magdagdag ng "Planuhin ang iyong biyahe" na link box sa iyong mga artikulo tungkol sa Portal Bridge.
  • Mag-link sa mga partikular na tool batay sa nilalaman ng artikulo.
  • Mayroon kaming OG image at social card na handa na para sa maayos na preview sa social media.
  • Inirerekomendang anchor text: "Gamitin ang libreng Reroute NJ tool para planuhin ang iyong biyahe."

Mag-ambag ng balita

  • Nagko-curate kami ng balita tungkol sa Portal Bridge sa aming pahina ng mga balita.
  • Para magsuhestyon ng artikulo, mag-bukas ng GitHub issue na may URL at mga detalye.
  • O mag-email sa amditisj@montclair.edu na may "Reroute NJ coverage" sa subject line.
  • Kasama namin ang lahat ng kaugnay na balita — opinyon, pagsusuri, community reporting, at iba pa.

Co-branding at customization

Gusto mo bang patakbuhin ang Reroute NJ gamit ang sarili mong branding? Dahil open source ito, kaya mo.

Opsyon 1: I-fork ang proyekto

1

I-fork ang GitHub repository sa account ng iyong organisasyon.

2

I-edit ang css/styles.css para baguhin ang kulay, font, at branding. Lahat ng theme value ay CSS custom property sa :root block.

3

Palitan ang pangalan ng brand, logo, at footer credit ng sarili mo. Magdagdag ng header/nav ng iyong newsroom kung gusto mo.

4

I-deploy sa GitHub Pages, Netlify, Vercel, o sarili mong server. Hindi kailangan ng build step — static HTML lang ito.

Opsyon 2: I-embed na may sariling intro

1

I-embed ang tool gamit ang iframe code sa itaas.

2

Magdagdag ng sarili mong header, intro text, at branding sa paligid ng iframe.

3

I-style ang paligid na pahina para tumugma sa disenyo ng iyong publikasyon.

Paano mag-ambag

Ang Reroute NJ ay isang proyekto ng komunidad. Narito kung paano ka makakatulong, anuman ang iyong antas ng teknikal na kasanayan.

Mga hindi teknikal na kontribusyon

  • Magsuhestyon ng artikulo: May alam kang artikulo tungkol sa Portal Bridge na hindi namin nabanggit? Mag-bukas ng issue o mag-email sa joe@amditis.com.
  • Mag-ulat ng pagkakamali: Kung mali ang datos ng iskedyul o oras ng transfer, ipaalam sa amin.
  • Ibahagi ang tool: Sabihin sa iyong mga kaibigang commuter, ibahagi sa social media, i-post sa iyong neighborhood group.
  • Magsalin: Tulungan kaming isalin ang tool sa Espanyol o iba pang wika ng mga sakay ng NJ Transit.

Mga teknikal na kontribusyon

  • Mag-ulat ng bug: Nakakita ng bug? Mag-bukas ng issue sa GitHub.
  • Mag-submit ng pull request: Ang codebase ay vanilla HTML/CSS/JS na walang build step. I-fork, baguhin, mag-PR.
  • Magdagdag ng datos: Tulungan kaming panatilihing updated ang data/coverage.json ng mga bagong artikulo.
  • Pagbutihin ang accessibility: Screen reader testing, WCAG compliance, keyboard navigation.
